# Redis

> 1、基础知识：redis 中的对象类型有哪些，底层实现用了哪些数据结构？

remote dictionary series 远程字典序列

内存数据库，kv 数据库，数据结构数据库，包括以下 5 种数据类型，

- `string` 这里是支持二进制结构安全的字符串
- `list` 链表（双向）
- `set` 集合（无序唯一，但实现的时候有序）
- `hash` 哈希
- `zset` 有序集合，加上一个用于排序的字段

其他的数据结构：

- `stream` 消息队列流式结构
- `hyberloglog` 统计相关
- ……可供选择的数据结构



内部实现原理，

key-val 基于 hash，val 又有几种不同的实现方法，

- hash, key-val 表示 val `O(1)`
- 双向链表 `O(n)`
- 压缩列表 `O(n)`
- 跳表 `O(log(n))`：多层级的有序链表
- 整数数组 `O(1)`：用于实现无序集合 set，集合常见的操作为：交并差，使用归并排序进行排序
- 动态字符串
