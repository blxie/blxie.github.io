# 编码风格

## 命名规则

变量：全部小写（用下划线进行分割），成员属性以 `m_` 开头或者以 `_` 结尾

类型名（类/结构体/枚举）：驼峰命名法，每个单词首字母大写

枚举：字母全大写，用 `_` 分隔

函数：第一个单词首字母小写，其余单词首字母大写



## 条件判断

> 如果有多个条件判断，将常量值放在之前，否则将常量值放在后面

```c
if ('e' == *ptr || 'E' == *ptr) {
    ptr++;
    if ('+' == *ptr || '-' == *ptr) {
        ptr++;
    }

    if (!ISDIGIT029(*ptr)) {
        return LEPT_PARSE_INVALID_VALUE;
    }

    for (ptr++; ISDIGIT029(*ptr); ptr++);
}
```





## ++i & i++

> 一般 指针/int 使用后置 `(int/ptr)++`，`++OBJ`，这是因为后置 ++ 会产生临时对象，影响效率！
>
> 不过还是要根据实际情况进行具体的选择！
