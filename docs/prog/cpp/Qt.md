下载安装包：https://download.qt.io/official_releases/online_installers/

不要直接运行！！！在命令行中添加 mirror，注意不是 mirrors！

```
--mirror https://mirrors.aliyun.com/qt
--mirror https://mirrors.tuna.tsinghua.edu.cn/qt
```
